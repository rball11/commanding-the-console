# Commanding the Console 

## Tips and Tricks

- Making Commits
    - Commits should be short and meaningful
    - Make frequent commits
- VS Code Keyboard Shortcuts
    - **ctrl** + **'** for toggling the terminal
    - **ctrl** + **k** + **s** to "Save All"
